## Perché Minions?

<CardGrid stagger>
  <Card title="Tre primitive semplici" icon="pencil">
    Tutto è costruito da **Minions** (istanze), **MinionTypes** (schemi) e **Relations** (collegamenti tipizzati).
  </Card>
  <Card title="Complessità progressiva" icon="add-document">
    Inizia con un singolo agente piatto. Cresci verso strutture annidate con memoria, competenze e team — senza necessità di migrazione.
  </Card>
  <Card title="Validazione basata su schema" icon="approve-check">
    Ogni campo viene validato rispetto al suo schema di tipo. Niente più incertezze sulla forma dei dati del tuo agente.
  </Card>
  <Card title="Agnostico rispetto al framework" icon="setting">
    Funziona con qualsiasi backend di archiviazione, qualsiasi runtime, qualsiasi framework IA. TypeScript e Python puri, senza opinioni sull'infrastruttura.
  </Card>
</CardGrid>
