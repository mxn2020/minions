## Warum Minions?

<CardGrid stagger>
  <Card title="Drei einfache Primitive" icon="pencil">
    Alles basiert auf **Minions** (Instanzen), **MinionTypes** (Schemas) und **Relations** (typisierte Verknüpfungen).
  </Card>
  <Card title="Progressive Komplexität" icon="add-document">
    Beginnen Sie mit einem einzelnen flachen Agenten. Wachsen Sie zu verschachtelten Strukturen mit Gedächtnis, Fähigkeiten und Teams — keine Migration nötig.
  </Card>
  <Card title="Schema-gesteuerte Validierung" icon="approve-check">
    Jedes Feld wird gegen sein Typschema validiert. Kein Rätselraten mehr über die Form Ihrer Agentendaten.
  </Card>
  <Card title="Framework-agnostisch" icon="setting">
    Funktioniert mit jedem Storage-Backend, jeder Laufzeitumgebung, jedem KI-Framework. Reines TypeScript & Python, keine Meinungen zur Infrastruktur.
  </Card>
</CardGrid>
