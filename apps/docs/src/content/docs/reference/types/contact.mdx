## Example

{

## Usage

<Tabs syncKey="lang">
  <TabItem label="TypeScript">
  ```typescript
  import { TypeRegistry, createMinion } from 'minions-sdk';

  const registry = new TypeRegistry();
  const type = registry.getBySlug('contact')!;

  const { minion, validation } = createMinion({
    title: 'Jane Smith',
    fields: { name: 'Jane Smith', email: 'jane@example.com' },
  }, type);
  ```
  </TabItem>
  <TabItem label="Python">
  ```python
  from minions import TypeRegistry, create_minion

  registry = TypeRegistry()
  type_ = registry.get_by_slug("contact")

  minion, validation = create_minion(
      {"title": "Jane Smith", "fields": {"name": "Jane Smith", "email": "jane@example.com"}},
      type_,
  )
  ```
